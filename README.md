**CAUTION!! EXPERIMENTAL REPOSITORY**

**yurie** is a cloud-based music player.

## Prerequisites

- Node.js (v0.10.x)
- gulp

## Building

```sh
$ npm install
$ npm run bower
$ npm run build
```

## Developing

```sh
$ npm install -g gulp
$ gulp watch
$ gulp webserver
```
